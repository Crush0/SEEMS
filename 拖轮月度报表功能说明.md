# 拖轮月度报表功能说明

## 功能概述

拖轮月度报表功能允许用户生成和导出拖轮的月度能效报表为PDF格式。

## 功能特性

### 1. 统计数据展示
- **总作业艘次**：月度总作业次数
- **总作业小时**：月度总作业时长（小时）
- **单次平均作业时长**：每次作业的平均时长（小时）
- **累计航行里程**：月度累计航行距离（海里）

### 2. 能耗统计表格
报表包含以下能耗数据：

| 项目 | 数值 | 备注/单位 |
|------|------|-----------|
| 月度总耗电量 | 45,800 | kWh |
| 单船平均能耗 | 243 | kWh/艘次 |
| 每海里能耗 | 48.2 | kWh/海里 |

### 3. 每日能耗折线图
展示月度每日能耗变化趋势，使用ECharts可视化。

### 4. 每日详细数据表格
包含每日的能耗、作业艘次、航行里程等详细数据。

### 5. PDF导出功能
支持将整个报表导出为PDF文件，便于存档和分享。

## 使用方法

### 前置要求

需要安装以下npm包：

```bash
cd seems-frontend
npm install html2canvas jspdf --save
```

### 操作步骤

1. **进入报表管理页面**
   - 登录系统后，在左侧菜单中选择"报表管理" -> "拖轮月度报表"

2. **选择报表参数**
   - 选择年份
   - 选择月份
   - 船舶信息会自动获取当前用户关联的船舶

3. **生成报表**
   - 点击"生成报表"按钮
   - 系统将根据选择的参数生成月度报表

4. **查看报表**
   - 统计数据卡片：显示关键指标
   - 能耗统计表格：显示详细的能耗数据
   - 每日能耗折线图：可视化展示能耗趋势
   - 每日详细数据表格：显示每日的具体数据

5. **导出PDF**
   - 点击"导出PDF"按钮
   - 系统将自动生成PDF文件并下载
   - 文件名格式：拖轮月度报表_[船舶名称]_[年份]年[月份]月.pdf

## 数据计算逻辑

### 统计指标计算

1. **总作业艘次**：统计该月所有航次数量
2. **总作业小时**：所有航次的航行时间总和（秒转小时）
3. **单次平均作业时长**：总作业小时 / 总作业艘次
4. **累计航行里程**：所有航次的航行距离总和
5. **月度总耗电量**：所有航次的电能消耗总和
6. **单船平均能耗**：月度总耗电量 / 总作业艘次
7. **每海里能耗**：月度总耗电量 / 累计航行里程

### 数据来源

- **航次数据**：`voyage_log` 表
- **GPS数据**：`gps_log` 表
- **船舶信息**：`ship_base_info` 表

## 注意事项

1. **数据完整性**：确保数据库中有完整的月度数据
2. **PDF导出**：首次使用需要安装html2canvas和jspdf依赖
3. **浏览器兼容性**：推荐使用Chrome或Edge浏览器以获得最佳体验
4. **数据权限**：用户只能查看自己关联船舶的报表数据
